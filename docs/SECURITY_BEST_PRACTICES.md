# Best Practice Security untuk Repo

## Checklist PR Sehat
- Pastikan semua pengujian lulus
- Sertakan deskripsi yang jelas untuk setiap perubahan
- Jangan sertakan perubahan yang tidak relevan

## Review
- OTP (One Time Password) untuk akses kritikal
- Verifikasi identitas reviewer

## Branch Protection
- Aktifkan perlindungan branch untuk main
- Pastikan at least satu review sebelum merge

## Incident Reporting
- Laporkan kejadian anomali ke tim keamanan
- Sertakan informasi detail dan langkah-langkah reproduksi
